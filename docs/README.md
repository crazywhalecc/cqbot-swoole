# 快速上手

## 安装框架和环境

此命令可一键以模板安装框架！（仅限 Linux 和 macOS）

```bash
bash <(curl -fsSL https://zhamao.xin/v3.sh)
```

## 运行框架

```bash
cd zhamao-v3/
./zhamao server
```
